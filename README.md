# 💰 Personal Finance Tracker Pro

A modern, responsive personal finance management application built with **HTML, CSS, and Vanilla JavaScript** that helps users track income, expenses, savings, and financial activity in real time.

---

## 🚀 Live Features

### 📊 Financial Dashboard

* Real-time balance calculation
* Total income tracking
* Total expense tracking
* Transaction statistics overview
* Largest expense monitoring

### 💳 Transaction Management

* Add new transactions
* Delete individual transactions
* Clear all transactions
* Automatic transaction date tracking
* Newest transactions displayed first

### 🔍 Smart Filtering

* Search transactions instantly
* Filter by:

  * All Transactions
  * Income
  * Expenses

### 💾 Persistent Storage

* Browser LocalStorage integration
* Data remains available after page refresh
* No backend required

### 🎨 Modern User Experience

* Professional dashboard UI
* Responsive mobile-first design
* Smooth animations and hover effects
* Dark mode support structure
* Clean and intuitive interface
