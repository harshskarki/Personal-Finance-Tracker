## ✨ Features

### 📊 Financial Dashboard

* Real-time balance calculation
* Total income tracking
* Total expense tracking
* Savings calculation
* Largest expense monitoring
* Transaction statistics overview

### 💳 Transaction Management

* Add income and expense transactions
* Delete individual transactions
* Clear all transactions
* Automatic date tracking
* Newest transactions displayed first

### 🔍 Search & Filtering

* Instant transaction search
* Filter by income
* Filter by expenses
* View all transactions

### 🌙 Modern User Experience

* Responsive design for desktop, tablet, and mobile
* Clean dashboard interface
* Smooth animations and transitions
* Dark mode support
* Interactive UI components

### 💾 Persistent Storage

* Browser LocalStorage integration
* Automatic data persistence
* No backend required
* Data remains after page refresh

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/harshskarki/Personal-Finance-Tracker.git
```

Navigate to the project folder:

```bash
cd Personal-Finance-Tracker
```

Open the application:

```bash
index.html
```

No frameworks, build tools, or additional dependencies are required.
